About Islamic Portal

The Islamic Portal began as a lightweight, privacy-focused project to make essential Islamic resources easily accessible across devices. It combines daily prayer times, Quranic passages, hadith references, and a curated resources index — all presented with a modern, responsive UI inspired by cultural aesthetics.

Mission

To provide accurate, easy-to-use Islamic tools and educational materials that respect user privacy and work well on low-bandwidth devices and phones.

Features

  • Geolocation-aware prayer times with live countdown
  • Themeable UI (Ghanaian kente / Islamic architecture)
  • Curated resources with live search and easy access
  • Minimal dependencies — pure HTML/CSS/JS for speed

Privacy

This site does not store personal information by default. The newsletter form is a mock in this demo; if you connect a backend, consider storing emails securely and providing unsubscribe options.

Contribute

To contribute resources, translations, or UI improvements, fork the repo and submit a pull request. Keep content verifiable and cite sources for religious texts.
